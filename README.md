## Book Shop Projesi

Bu proje kullanıcıların kitapları isteklerine göre filtreleyip listeleme yapmasını amaçlanmaktadır. Kullanıcı dostu arayüzü ile her ekrana uygun olacak şekilde tasarlanmıştır. Kitapların sepete eklenmesi, sayısının artırılması-azaltılması ve silme işlemi yapılabilmektedir.

## Kullanılan Teknolojiler

-HTML

-CSS

-JS
